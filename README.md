# Weather-Diary
Программа для записи дневника погоды
# Название проекта Weather-Diary
**Автор:** Елизавета Недолужко
**Вариант:** Weather Diary
**Дата сдачи:** 04.05.2026
## Описание программы 
Графическое приложение на Python для ведения дневника наблюдений за погодой. Приложение позволяет добавлять, просматривать, фильтровать и удалять записи о погоде, а также сохранять и загружать данные в формате JSON.
## Требования для запуска 
Что нужно установить на компьютер: 
- Python 3.10 или выше 
- Библиотеки: `pip install tkinter`
## Как запустить 
```bash 
git clone https://github.com/xyz-dsgn/Weather-Diary
cd Weather-Diary
python main.py
```
## Как пользоваться 
Добавление записи о погоде
- Введите дату в формате ГГГГ-ММ-ДД (например, 2026-05-04)
- Введите температуру в градусах Цельсия (число)
- Введите описание погоды (например, "Солнечно, тепло")
- Отметьте наличие осадков (Да/Нет)
- Нажмите кнопку "Добавить запись"
## Пример работы 
1. Запуск программы — открывается окно с полями ввода и таблицей.
2. Добавление записи о погоде:
 - Дата: 2026-05-04
 - Температура: 18.5
 - Описание: Солнечно, легкий ветер
 - Осадки: Нет
 - Нажать Добавить запись
3. Фильтр по дате:
 - Ввести 2026-05-04 → нажать Применить фильтр — показываются только записи за указанную дату
 - Фильтр по температуре:
 - Ввести 15 (выше 15°C) → нажать Применить фильтр — остаются записи с температурой выше +15°C
4. Комбинированный фильтр:
 - Ввести дату 2026-05-04 и температуру выше 10 → нажать Применить фильтр — показываются записи за 04.05.2026 с температурой выше 10°C
